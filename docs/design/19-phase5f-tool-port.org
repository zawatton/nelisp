* §0 要旨

Phase 5-F.1 で anvil-state SQLite KV port + state-* MCP tool 6 本 expose
(tools/list 7 → 13) が完了。本 sub-phase 5-F.2 では anvil-manifest
*headless profile* (~30 tool) のうち、anvil.el 本体 dep 無しで pure
Elisp 実装可能な 11 tool を NeLisp 側に port。tools/list は 13 → 24 に拡大。

非 Emacs user (Stage D launcher 経由) からの daily file/git ops が
NeLisp MCP 単体でカバーされる中核拡張。

* §1 motivation

** 1.1 現状 gap

NeLisp MCP server は Phase 5-E (7 tool) + 5-F.1.2 (6 state tool) で
13 tool。しかし anvil headless profile 想定 ~30 tool のうち、daily 必須:
- file replace / insert / delete / append / snippet read
- git diff / HEAD / branch / repo-root
- data store list / delete

の 11 tool が未 expose。Stage D Phase 6.1 launcher を非 Emacs user に
release した時点で「ファイル書き換え」「git 状態」が tools/call で
できないと dogfood 価値が低い。

** 1.2 Phase 5-F.2.0 sub-phase 想定 (本 doc では skip)

5-F.1.0 と異なり 5-F.2 には primitive 追加 pre-req 不要 — Phase 5-C で
file / process / regex 系 primitive は揃い済。すぐ 5-F.2.1 から実装可能。

* §3 tool 表 (11 本)

** 5-F.2.1 file ops (5 本)

| tool | input | output | 備考 |
|------+-------+--------+------|
| file-replace-string | path, old, new | :replaced N | ファイル全文 substring 置換 |
| file-insert-at-line | path, line, content | :inserted t | 1-indexed 行に挿入 |
| file-delete-lines | path, start, count | :deleted N | start から count 行削除 |
| file-append | path, content | :appended N | 末尾追加、改行 auto |
| file-read-snippet | path, line, window | :start L :end L :content | 行 L 中心の window 行 |

** 5-F.2.2 git ops (4 本)

| tool | input | output | 備考 |
|------+-------+--------+------|
| git-diff-names | rev? | :files [...] | rev 省略で uncommitted、指定で since rev |
| git-head-sha | (none) | :sha | 現 HEAD の full SHA |
| git-branch-current | (none) | :branch | symbolic ref short |
| git-repo-root | (none) | :root path | git rev-parse --show-toplevel |

** 5-F.2.3 data ops (2 本)

| tool | input | output | 備考 |
|------+-------+--------+------|
| data-list-keys | path? | :keys [...] | path 省略で top-level、指定で nested |
| data-delete-path | path | :deleted t | 既存 keypath を nil 化 (実質 unset) |

合計 *11 tool*、tools/list 13 → 24。

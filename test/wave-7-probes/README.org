#+TITLE: Wave 7 standardized probe library
#+STATUS: subagent / claude main 共用

Wave 7 (Doc 49 §7) 関連の subagent 反復ループ短縮のため、common
probe を `.el` ファイル化。subagent prompt から「`./target/release/nelisp
--batch -l <abs-path>` で呼べ」と参照する。

| Probe              | 用途                                         | typical 所要      |
|--------------------+----------------------------------------------+-------------------|
| =probe-single.el=  | type-of / prin1 / puthash の最小 Record sanity | < 1s              |
| =probe-chain.el=   | require chain (5 features) hang/OK 判定        | < 2s (R6h regress = hang) |
| =probe-first-n.el= | first N manifest entries (default 10)        | ~5-30s            |
| =probe-208.el=     | full 208 manifest pass rate (`PASS RATE: x/y`) | 60-600s           |
| =probe-stress.el=  | fset loop で latent .o hang trigger 確認        | < 30s (R6h hang trigger 検出に) |

すべて =NELISP_PROBE_REPO=/abs/path= で repo root 指定。未設定時 default
= =/home/madblack-21/Cowork/Notes/dev/nelisp=。

* 反復実験 workflow

1. 単体: =probe-single.el= で Record op の基本動作 (= < 1s)
2. require chain: =probe-chain.el= で hang 検出 (= < 2s)
3. small scale: =probe-first-n.el= NELISP_PROBE_LIMIT=10 で進捗 metric (= < 30s)
4. **着地時のみ**: =probe-208.el= で full pass rate
5. regression 専用: =probe-stress.el= NELISP_STRESS_N=20 で trigger 境界

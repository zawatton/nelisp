#+TITLE: nelisp-regex Package

* Summary

=nelisp-regex= is an extracted utility library package for the NeLisp
package split.  It provides a pure Elisp subset POSIX/Emacs regex
engine for NeLisp text and buffer consumers without consulting host
Emacs regex primitives at run time.

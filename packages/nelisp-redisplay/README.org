#+TITLE: nelisp-redisplay Package

* Summary

=nelisp-redisplay= is an extracted NeLisp library package per Doc 45. It
provides terminal redisplay, dirty-row tracking, and window helpers on
top of =nelisp-buffer=.

* Layout

* Public API

- =nelisp-window=
- =nelisp-redisplay-clear-screen=

- =nelisp-window-mark-all-dirty=

#+TITLE: nelisp-server Package

* Summary

=nelisp-server= is an extracted NeLisp package per Doc 45. It provides
the MCP stdio server spine, JSON-RPC method registry, and tool
registration surface used by anvil-on-nelisp.

* Layout

#+begin_src text
packages/nelisp-server/
├── README.org
├── src/
│   └── nelisp-server.el
└── test/
    └── nelisp-server-test.el
#+end_src

* Dependencies

- Runtime dependency: NeLisp core =nelisp= v1.0 load-path environment
- Elisp dependency declared by the library: =cl-lib=
- External host-Elisp dependency: =json=
- Internal NeLisp dependencies: none

#+TITLE: nelisp-secure-hash Package

* Summary

=nelisp-secure-hash= is an extracted utility library package for the
NeLisp package split.  It provides pure Elisp MD5, SHA-1, and SHA-256
digest functions for the standalone path without depending on host
Emacs =secure-hash= or =md5=.

* Public API

- =nelisp-hash-sha1=
- =nelisp-hash-sha256=
- =nelisp-hash-secure-hash=

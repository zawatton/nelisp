#+TITLE: nelisp-eventloop-multiplex Package

* Summary

=nelisp-eventloop-multiplex= provides the fd polling and callback
dispatch layer that extends =nelisp-eventloop= with selectable fd
multiplexing and event-source adapters.

* Layout

#+begin_src text
packages/nelisp-eventloop-multiplex/
├── README.org
├── src/
│   └── nelisp-eventloop-multiplex.el
└── test/
    └── nelisp-eventloop-multiplex-test.el
#+end_src

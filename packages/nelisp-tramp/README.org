#+TITLE: nelisp-tramp Package

* Summary

=nelisp-tramp= provides the explicit-stack trampoline evaluator for
NeLisp.  It preserves the observable semantics of =nelisp-eval= while
avoiding host stack growth during deep NeLisp execution.

* Public API

- =nelisp-tramp-eval=

* Dependencies

- Runtime dependency: NeLisp core =nelisp= v1.0 load-path environment
- Elisp dependencies declared by the library: =nelisp-eval=
- Soft macro dependencies declared via =declare-function=: =nelisp-macro=

* Dependents

- Source dependents: none under =src/=
- Test dependents: package-local =test/nelisp-tramp-test.el=

#+TITLE: nelisp-defs-index Package

* Summary

=nelisp-defs-index= provides a persistent SQLite-backed index of Elisp
definitions, references, signatures, and feature edges.  It is the
extracted NeLisp library package used to answer definition queries in a
single indexed lookup instead of a grep walk.

* Dependencies

- Runtime dependency: NeLisp core =nelisp= v1.0 load-path environment
- Elisp dependencies declared by the library: =cl-lib=, =subr-x=
- Host capability dependency: built-in Emacs SQLite support
- Internal NeLisp package dependencies declared by the library: none

* Dependents

- Runtime dependents in =src/=: =nelisp-tools=
- Test dependents in =test/=: =nelisp-defs-index-test.el=,
  =nelisp-indices-fix-test.el=
- Existing callers continue using =(require 'nelisp-defs-index)=; the
  root build load-path resolves the extracted package location.

#+TITLE: nelisp-sys language reference
#+OPTIONS: toc:2

* Overview

=nelisp-sys= is a typed systems subset.  This document is the package-local
language reference (Doc 130 extraction criterion 8).  It is derived from
design Doc 131 and is the authority once the package is extracted.

* Source namespace

Source forms use the =sys:= prefix.  The compiler API uses =nelisp-sys-=.

* Top-level forms

- =(sys:defstruct NAME (:repr c|sys) (FIELD TYPE)...)= — struct definition.
- =(sys:defun NAME ((ARG TYPE)...) RET (ATTRS...) BODY...)= — function.
- =(sys:extern NAME (:symbol S :abi c :unsafe t) (ARGS...) RET (ATTRS...))=
  — imported C function declaration.
- =(sys:const NAME TYPE VALUE)= — compile-time constant.
- =(sys:static-assert FORM MSG)= — compile-time assertion.

* Body forms (MVP)

- arithmetic: =+ - * / mod=, comparisons =< <= > >= = /==, =and or not=.
- =(let ((NAME TYPE EXPR)...) BODY...)= — typed locals.
- =(if C THEN ELSE)=, =(cond ...)=, =(while C BODY...)=, =(seq ...)=.
- =(sys:load-field PLACE FIELD)= / =(sys:store-field! PLACE FIELD VALUE)=.
- =(sys:slice-len S)= / =(sys:slice-ref S I)= / =(sys:slice-set! S I V)=.
- =(sys:cast TYPE EXPR)= — explicit width/sign conversion.
- =(sys:unsafe (:reason "...") BODY...)= — unsafe boundary.
- =(sys:with-borrow ((P (& T) PLACE)) BODY...)= and =-mut= variant.
- =(sys:sizeof TYPE)= / =(sys:alignof TYPE)= / =(sys:offsetof STRUCT FIELD)=.

Detailed semantics live in =memory-safety.org= and design Doc 131.

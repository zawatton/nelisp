#+TITLE: nelisp-sqlite Package

* Summary

=nelisp-sqlite= provides the NeLisp SQLite substrate with an Emacs
30-style =sqlite-*= surface backed by the Rust-built Emacs module and
runtime shared library.  It is the extracted NeLisp package for the
SQLite FFI bridge.

* Layout

├── src/
│   └── nelisp-sqlite.el

#+end_src

* Dependencies

- Runtime dependency: NeLisp core =nelisp= v1.0 load-path environment
- Elisp dependency declared by the library: =cl-lib=
- Internal NeLisp package dependency declared by the library:
  =nelisp-json=
- Built artifact dependencies:
  =nelisp-runtime/target/release/libnelisp_runtime.so= and
  =nelisp-runtime/target/release/nelisp-sqlite-module.so=

* Dependents

- Runtime dependents in =src/=: none found under current =src/=
- Test dependents in =test/=: =nelisp-sqlite-test.el=

  build load-path resolves the extracted package location.

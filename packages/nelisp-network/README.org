#+TITLE: nelisp-network Package

* Summary

=nelisp-network= provides the NeLisp network and HTTP substrate:
network process wrappers, actor/event posting, and small HTTP GET/HEAD
helpers with cache support.

* Layout

#+begin_src text
packages/nelisp-network/
├── README.org
├── src/
│   └── nelisp-network.el
└── test/
    └── nelisp-network-test.el
#+end_src

* Public API

- =nelisp-open-network=
- =nelisp-network-send-string=
- =nelisp-network-close=
- =nelisp-network-live-p=
- =nelisp-network-list=
- =nelisp-network-get=
- =nelisp-network-put=
- =nelisp-http-cache-clear=
- =nelisp-http-cache-size=
- =nelisp-http-get=
- =nelisp-http-head=

* Dependencies

- Runtime dependency: NeLisp core =nelisp= v1.0 load-path environment
- Elisp dependencies declared by the library: =cl-lib=, =nelisp-eval=,
  =nelisp-actor=, host Emacs =url=

* Dependents

- Source dependents: anvil.el modules via fboundp delegation (architecture α; NeLisp side has no consumers — Wave 1 deleted =nelisp-tools=)
- Test dependents: package-local =test/nelisp-network-test.el=

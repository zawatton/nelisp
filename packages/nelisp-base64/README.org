#+TITLE: nelisp-base64 Package

* Summary

=nelisp-base64= is an extracted utility library package for the NeLisp
package split.  It provides a pure Elisp RFC 4648 base64
encoder/decoder for the standalone path without depending on host
Emacs base64 primitives.

* Layout

#+end_src

* Public API

- =nelisp-b64-encode-string=

- =nelisp-b64-decode-string=

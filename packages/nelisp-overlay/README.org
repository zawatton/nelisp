#+TITLE: nelisp-overlay Package

* Summary

=nelisp-overlay= is an extracted NeLisp library package per Doc 45. It
provides overlay storage, querying, mutation, and Emacs-compat overlay
APIs for the Phase 9c text-property stack.

* Layout

- Elisp dependency declared by the library: =cl-lib=
- Internal NeLisp dependencies loaded dynamically: =nelisp-textprop-keymap=

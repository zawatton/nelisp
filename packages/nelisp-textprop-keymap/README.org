#+TITLE: nelisp-textprop-keymap Package

* Summary

=nelisp-textprop-keymap= is an extracted NeLisp library package per Doc
45. It provides text-property and overlay keymap chain injection,
provider registration, and lookup helpers for the Phase 9c stack.

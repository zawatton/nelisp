#+TITLE: nelisp-jit Package

* Summary

=nelisp-jit= is an extracted NeLisp package per Doc 45. It provides the
host-Elisp JIT compiler path for NeLisp lambdas, with fallback to the
existing evaluator and bytecode runtime for unsupported forms.

* Layout

#+begin_src text
packages/nelisp-jit/
├── README.org
├── src/
│   └── nelisp-jit.el
└── test/
    └── nelisp-jit-test.el
#+end_src

* Public API

- =nelisp-jit-enabled=
- =nelisp-jit-try-compile-lambda=
- =nelisp-jit-refresh-primitive-set=

* Dependencies

- Runtime dependency: NeLisp core =nelisp= v1.0 load-path environment
- Elisp dependency declared by the library: =cl-lib=
- Internal NeLisp dependency: =nelisp-eval=

* Dependents

- Direct source dependents: none under =src/=
- Test dependents: =packages/nelisp-jit/test/nelisp-jit-test.el=
